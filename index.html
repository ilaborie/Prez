What's Prez

Prez is just a basic HTML slideshow system


Prez is build on modern web components: HTML5, CSS3.
The dynamic part is written in CoffeeScript.
It's also use great JavaScript libraries: Zepto.js, Moment.js, Prism, Hammer.js

Why Prez ?

I've just want to create my own slideshow system

Why create another HTML5 slideshow system?
Just for fun!

Igor

Smartphone & Tablet support

Prez support mobile devices with tap and swipe events.

Thanks to Hammer.js

To avoid move with mouse click just remove the hammer.js from index.html.

Style your slides


        

            Prez use Less to create CSS files.
            Just update less file to change the slideshow style.
        

        

            
main.less

            
Assemble others less files, include


            
theme/*.less

            
Some themes, create yours here and include it into main.less


            
prez.less

            
Standard presentation style and behaviour


            
print.less

            
Create a specific css for printer


            
prism.less

            
Style your <code> block


            
talk.less

            
Custom style for your current slideshow


            
variable.less

            
Some variables and useful mixins
