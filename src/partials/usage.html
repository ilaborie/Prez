Build a slideshow

Requirements (do once)

  1. If necessary, install Node, Grunt, Bower
  2. git clone the project
  3. run npm install to load build dependencies

Daily build

  1. run grunt build the presentation
  2. run grunt dev to develop the presentation

Style your slides

Prez use Less to create CSS files. Just update less file to change the slideshow style.

main.less
Assemble others less files, include
theme/*.less
Some themes, create yours here and include it into main.less
prez.less
Standard presentation style and behaviour
print.less
Create a specific css for printer
prism.less
Style your <code> block
talk.less
Custom style for your current slideshow
variable.less
Some variables and useful mixins

Use without building process

  • Run grunt
  • Get target/script/prez.js
  • Include this script into you HTML file
  • Use a generated CSS file or create yours
