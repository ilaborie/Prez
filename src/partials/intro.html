What's Prez

Prez is just a basic HTML slideshow system


Prez is build on modern web components: HTML5, CSS3.
The dynamic part is written in CoffeeScript.
It's also use great JavaScript libraries: Zepto.js, Moment.js, Prism, Hammer.js

Why Prez ?

I've just want to create my own slideshow system

Why create another HTML5 slideshow system?

Smartphone & Tablet support

Prez support mobile devices with tap and swipe events.

Thanks to Hammer.js
